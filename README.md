# TrueRAG

TrueRAG is a High-Performance Modular Monolith **.NET 10** platform for Retrieval-Augmented Generation (RAG) and enterprise search. It serves as the reference architecture and marketing showcase for **TrueParser** (a High-Fidelity Parsing Engine) and runs through a single executable host (`TrueRag.Host`).

By leveraging *structural fidelity* (preserving tables, document hierarchy, and paragraphs) rather than standard token-based chunking, TrueRAG unlocks advanced retrieval features like Structural Expansion, Multi-Hop linking, and version-aware Structural Diffing.
