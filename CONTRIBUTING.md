# Contributing

TrueRAG is currently maintained as an internal-first repository.

## Policy

- External code contributions are not accepted at this time.
- Please do not open pull requests with code changes unless you have been explicitly invited by the maintainers.
- Issue reports, documentation suggestions, feature requests, and security disclosures are welcome.

## Why

TrueRAG has strict architectural boundaries around tenancy, retrieval isolation, WAL safety, and background processing. Unvetted code changes can unintentionally weaken those guarantees.

## Future

If TrueRAG becomes a public SaaS or opens a contributor program later, this policy may be revised.
